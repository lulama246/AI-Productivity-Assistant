# 🌸 Lulu's Helping Hands

## Project Overview

**Lulu's Helping Hands** is an AI-powered workplace productivity and community support web application designed to assist individuals, volunteers, and organizations. The platform combines Artificial Intelligence with community engagement to simplify everyday tasks such as writing professional emails, summarizing meeting notes, planning tasks, conducting research, and providing instant support through an AI chatbot.

The project was developed as part of the **CAPACITI AI Skill Accelerator Programme** to demonstrate practical AI application, prompt engineering, workplace productivity, and responsible AI practices.

The goal of the platform is to empower people by making assistance more accessible while improving productivity through AI-driven automation.

---

## Features

### 🤖 AI Email Generator
- Generates professional emails.
- Supports Formal, Friendly, and Persuasive tones.
- Creates emails for sponsorship requests, donations, volunteer recruitment, and general communication.

### 📝 AI Meeting Notes Summarizer
- Summarizes lengthy meeting notes.
- Extracts key discussion points.
- Identifies action items, responsibilities, and deadlines.

### 📅 AI Task Planner
- Creates daily and weekly schedules.
- Prioritizes important tasks.
- Helps volunteers and staff organize their workload efficiently.

### 🔍 AI Research Assistant
- Summarizes articles and research topics.
- Provides key insights and recommendations.
- Simplifies complex information for quick understanding.

### 💬 Lulu AI Chat Assistant
- Answers questions about the platform.
- Assists users with volunteering, donations, and community support.
- Provides guidance on available services.

### ❤️ Community Support
- Volunteer registration.
- Donation section.
- Contact page.
- Information about community projects and services.

### 📱 Responsive Design
- Mobile-friendly interface.
- Tablet and desktop compatibility.
- Modern Purple and Pink user interface.

### 🔒 Responsible AI
- Encourages users to verify AI-generated information.
- Promotes ethical AI usage.
- Protects user privacy.

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/lulus-helping-hands.git
```

### 2. Navigate to the Project Folder

```bash
cd lulus-helping-hands
```

### 3. Install Dependencies (if applicable)

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

or if using a simple HTML project:

```bash
Open index.html in your web browser.
```

### 5. Build for Production (Optional)

```bash
npm run build
```
